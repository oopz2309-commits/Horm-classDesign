# 中山文旅美食导览 App 后端项目创建与环境配置

## 一、本阶段目标

本阶段主要完成 Spring Boot 后端项目的创建和基础环境配置，为后续开发用户登录、景点查询、美食查询、收藏和评论接口做准备。

本阶段需要完成以下内容：

1. 创建 Spring Boot 后端项目。
2. 配置 Maven 依赖。
3. 配置 MySQL 数据库连接。
4. 创建项目基础包结构。
5. 创建统一返回结果类 Result。
6. 创建数据库和数据表。
7. 启动项目，确认后端服务可以正常运行。

---

# 二、创建 Spring Boot 项目

## 1. 项目信息

建议使用 IntelliJ IDEA 创建 Spring Boot 项目。

项目基本信息如下：

| 配置项          | 内容                |
| ------------ | ----------------- |
| Project      | Maven             |
| Language     | Java              |
| Spring Boot  | 2.7.x             |
| Group        | com.zs            |
| Artifact     | travel            |
| Name         | zs-travel-backend |
| Package name | com.zs.travel     |
| JDK          | JDK 8 或 JDK 17    |

项目名称建议：

```text
zs-travel-backend
```

## 2. 推荐选择的依赖

创建项目时可以选择以下依赖：

```text
Spring Web

MySQL Driver
Lombok
```

如果创建项目时没有选择，也可以后期在 `pom.xml` 中手动添加。

# 三、pom.xml 配置

项目创建完成后，检查 `pom.xml` 文件，建议内容如下：

```xml
<dependencies>
    <!-- Spring Web：用于开发 REST 接口 -->
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-web</artifactId>
    </dependency>

    <!-- MyBatis：用于数据库访问 -->
    <dependency>
        <groupId>org.mybatis.spring.boot</groupId>
        <artifactId>mybatis-spring-boot-starter</artifactId>
        <version>2.3.1</version>
    </dependency>

    <!-- MySQL 驱动 -->
    <dependency>
        <groupId>com.mysql</groupId>
        <artifactId>mysql-connector-j</artifactId>
        <scope>runtime</scope>
    </dependency>

    <!-- Lombok：简化实体类 getter、setter 代码 -->
    <dependency>
        <groupId>org.projectlombok</groupId>
        <artifactId>lombok</artifactId>
        <optional>true</optional>
    </dependency>

    <!-- 测试依赖 -->
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-test</artifactId>
        <scope>test</scope>
    </dependency>
</dependencies>
```

如果你的 Spring Boot 版本是 3.x，也可以使用，但建议课程设计阶段优先使用 **Spring Boot 2.7.x**，兼容性更稳定。

# 四、application.yml 配置

在：

```text
src/main/resources/application.yml
```

中配置数据库连接信息：

```yaml
server:
  port: 8080

spring:
  datasource:
    driver-class-name: com.mysql.cj.jdbc.Driver
    url: jdbc:mysql://localhost:3306/zs_travel_app?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai
    username: root
    password: 123456

mybatis:
  mapper-locations: classpath:mapper/*.xml
  type-aliases-package: com.zs.travel.entity
  configuration:
    map-underscore-to-camel-case: true
```

注意：

```text
username 和 password 要改成你自己电脑 MySQL 的账号和密码。
```

例如你的 MySQL 密码不是 `123456`，就要改成自己的密码。

# 六、创建数据表

执行以下 SQL，创建 5 张核心表。

## 1. 用户表

```sql
CREATE TABLE user (
    id INT PRIMARY KEY AUTO_INCREMENT COMMENT '用户ID',
    username VARCHAR(50) NOT NULL UNIQUE COMMENT '用户名',
    password VARCHAR(100) NOT NULL COMMENT '用户密码',
    nickname VARCHAR(50) DEFAULT NULL COMMENT '用户昵称',
    avatar VARCHAR(255) DEFAULT NULL COMMENT '用户头像地址',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '注册时间'
) COMMENT='用户表';
```

## 2. 景点表

```sql
CREATE TABLE scenic_spot (
    id INT PRIMARY KEY AUTO_INCREMENT COMMENT '景点ID',
    name VARCHAR(100) NOT NULL COMMENT '景点名称',
    cover_image VARCHAR(255) DEFAULT NULL COMMENT '景点封面图',
    video_url VARCHAR(255) DEFAULT NULL COMMENT '景点视频地址',
    address VARCHAR(255) DEFAULT NULL COMMENT '景点地址',
    description TEXT COMMENT '景点介绍',
    type VARCHAR(50) DEFAULT NULL COMMENT '景点类型',
    score DECIMAL(2,1) DEFAULT 5.0 COMMENT '推荐评分',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间'
) COMMENT='景点表';
```

## 3. 美食表

```sql
CREATE TABLE food (
    id INT PRIMARY KEY AUTO_INCREMENT COMMENT '美食ID',
    name VARCHAR(100) NOT NULL COMMENT '美食名称',
    cover_image VARCHAR(255) DEFAULT NULL COMMENT '美食封面图',
    description TEXT COMMENT '美食介绍',
    recommend_shop VARCHAR(100) DEFAULT NULL COMMENT '推荐店铺',
    address VARCHAR(255) DEFAULT NULL COMMENT '店铺地址',
    price DECIMAL(6,2) DEFAULT NULL COMMENT '人均价格',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间'
) COMMENT='美食表';
```

## 4. 收藏表

```sql
CREATE TABLE favorite (
    id INT PRIMARY KEY AUTO_INCREMENT COMMENT '收藏ID',
    user_id INT NOT NULL COMMENT '用户ID',
    target_id INT NOT NULL COMMENT '收藏对象ID',
    type VARCHAR(20) NOT NULL COMMENT '收藏类型：scenic或food',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '收藏时间'
) COMMENT='收藏表';
```

## 5. 评论表

```sql
CREATE TABLE comment (
    id INT PRIMARY KEY AUTO_INCREMENT COMMENT '评论ID',
    user_id INT NOT NULL COMMENT '用户ID',
    target_id INT NOT NULL COMMENT '评论对象ID',
    type VARCHAR(20) NOT NULL COMMENT '评论类型：scenic或food',
    content VARCHAR(500) NOT NULL COMMENT '评论内容',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '评论时间'
) COMMENT='评论表';
```

# 九、创建统一返回类 Result

在：

```text
src/main/java/com/zs/travel/common/Result.java
```

中创建统一返回结果类：

```java
package com.zs.travel.common;

import lombok.Data;

@Data
public class Result<T> {

    private Integer code;
    private String message;
    private T data;

    public Result() {
    }

    public Result(Integer code, String message, T data) {
        this.code = code;
        this.message = message;
        this.data = data;
    }

    public static <T> Result<T> success() {
        return new Result<>(200, "操作成功", null);
    }

    public static <T> Result<T> success(T data) {
        return new Result<>(200, "操作成功", data);
    }

    public static <T> Result<T> success(String message, T data) {
        return new Result<>(200, message, data);
    }

    public static <T> Result<T> error(String message) {
        return new Result<>(500, message, null);
    }
}
```

# 十三、常见问题

## 1. 数据库连接失败

检查 `application.yml` 中的数据库账号和密码：

```yaml
username: root
password: 123456
```

如果你的 MySQL 密码不是 `123456`，需要改成自己的密码。

---

## 2. 数据库不存在

如果报错提示找不到数据库，需要先执行：

```sql
CREATE DATABASE zs_travel_app DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
```

---

## 3. 端口被占用

如果 8080 端口被占用，可以把端口改成 8081：

```yaml
server:
  port: 8081
```

访问地址也要改成：

```text
http://localhost:8081/test
```

---

## 4. Lombok 报错

如果 `@Data` 报红，需要检查：

1. IDEA 是否安装 Lombok 插件。
2. `pom.xml` 是否添加 Lombok 依赖。
3. 是否开启注解处理器。

IDEA 设置路径：

```text
Settings -> Build, Execution, Deployment -> Compiler -> Annotation Processors
```

勾选：

```text
Enable annotation processing
```
